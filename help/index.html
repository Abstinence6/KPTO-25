1 Назначение программ

Программы, входящие в состав Комплекта принадлежностей для технического обслуживания электронных блоков CMU-25, VMU-25, SGU-25 (далее - блоки), предназначены   для   работы  на  ПЭВМ  (ПК)  в   операционных   системах Windows 7/8/10.

В состав Комплекта принадлежностей также входит блок КПТО-25.

1.1 Назначение программы инсталляции KPTO-25_install.exe

Программа KPTO-25_install.exe предназначена для установки программного обеспечения (ПО) КПТО-25 на ПК.

1.2 Назначение программы запуска КПТО-25 Authorize.exe

Программа Authorize.exe предназначена для запуска ПО КПТО-25 на ПК по команде пользователя.

1.3 Назначение программы менеджера рабочей станции Pcmngr.exe

Программа Pcmngr.exe предназначена для запуска и инициализации программ ПО КПТО-25, управления выполнением программ и синхронизации потоков данных.

1.4 Назначение программы чтения и ввода регулировок Regul.exe

Программа Regul.exe предназначена для считывания и ввода регулировок блоков, визуализации считанных данных на экране ПК, сохранения считанных регулировок в виде файла на ПК с возможностью внесения регулировок в блоки.

1.5 Назначение программы редактора списка параметров RedPar.exe

Программа RedPar.exe предназначена для редактирования списка параметров и градуировки измерительных каналов.

 

1.6 Назначение программы редактора окон MyShell.exe

Программа MyShell.exe предназначена для отображения и настройки оконных интерфейсов.

1.7 Назначение программы ведения журнала Journal.exe

Программа Journal.exe предназначена для ведения электронного журнала сообщений.

 2 Условия выполнения программ

2.1 Аппаратное обеспечение

ПО КПТО-25 предназначено для работы на ПК с конфигурацией не хуже, чем:

-                               процессор Intel Pentium двухъядерный 2,0 ГГц;

-               объем оперативной памяти 4 Гб;

-               жесткий диск емкостью 500 Гб;

-               TFT монитор с поддержкой разрешения full HD (1920×1080);

-                               клавиатура, манипулятор типа «мышь» (далее – мышь);

-                               устройство чтения и записи CD-RW дисков (не обязательно);

-                               интерфейс USB (не менее двух разъемов).

 

2.2 Системное программное обеспечение

ПО КПТО-25 функционирует  под  управлением ОС  семейства Microsoft Windows 7 и выше.

В разделе 3 отмечалось, что доступ к функциям и программам КПТО ограничивается в зависимости от уровня доступа, который  определяется паролем и аппаратным USB-ключом, входящим в состав КПТО.

 

Внимание! Данное РО ориентировано на пользователя «Разработчик ДВИГАТЕЛЯ» и предусматриваЕт наличие USB-ключа.

Все редакторы свойств компонентов содержат:

- поле  «Имя компонента», предназначенное для идентификации компонентов одного типа. По умолчанию значение этого поля отсутствует;

- поля, предназначенные для установки параметров расположения компонента. «Левый» - отступ компонента от левого края окна, «Верхний» - отступ компонента от верхнего края окна;

- поля, предназначенные для задания геометрических размеров компонента. «Ширина» - ширина компонента, «Высота» - высота компонента.

Программа предназначена для просмотра электронной базы сообщений о действиях персонала, системных и прочих отказах, сформировавшихся в процессе работы КПТО.

3.9.1 Просмотр базы сообщений

Электронная база сообщений загружается автоматически при запуске программы «Журнал». Сообщения в электронной базе хранятся в кодированном виде и не подлежат редактированию. Сообщения загружаются в порядке, обратном их поступлению в электронную базу.

Формат сообщения:

- Имя задачи ¾  имя задачи, которая посылает сообщение;

- Сообщение ¾ текст сообщения;

- Время - время формирования сообщения;

- Ошибки ¾ текст ошибок, возникших при формировании сообщения.

На панели управления расположены кнопки:  «Поиск»,  «Сохранить результаты поиска»,  «Просмотр таблиц»,  «Предварительный просмотр»,  «Печать результатов поиска»,  «Выйти из приложения», дублирующие функции кнопок полей управления сортировкой и поиском.

Ниже расположена панель поиска, позволяющая выполнять поиск по времени – «Время» и по тексту –  «Искать текст».

В центре окна  расположена панель сортировки, позволяющая выполнять сортировку по полю и по времени.

В нижней части окна расположена панель отображения результатов поиска и сортировки, в которой отображается список сообщений (слева) и окно, содержащее выделенное в списке сообщение (справа).

Для просмотра полного текста сообщения в окне просмотра сообщений выберите сообщение на панели отображения результатов поиска и сортировки. Выбранное (активное) сообщение подсвечивается синим цветом.

Текст для поиска сообщения вводится в поля под надписью «Искать текст».

Пользователь может осуществлять поиск по следующим значениям:

- «Имя задачи» - поиск в базе сообщений среди значений «Имя задачи»;

- «Сообщение» - поиск в базе сообщений среди значений «Текст сообщения»;

Каждое поле содержит переключатель режима поиска: «И» и «ИЛИ».

«И» означает, что запрос получает положительный ответ, если в сообщении найдены все слова для выбранного поля поиска.

«ИЛИ» - если в сообщении найдено хотя бы одно слово для выбранного поля поиска.

Кнопки «’И’ поля», «’ИЛИ’ поля» устанавливают аналогичную логическую связь между полями поиска.

«’И’ поля» означает, что запрос получает положительный ответ, если получены положительные ответы для каждого поля поиска.

«’ИЛИ’ поля» означает, что запрос получает положительный ответ, если получен положительный ответ хотя бы для одного поля поиска.

 

Для выполнения поиска среди сообщений, полученных в результате предыдущего запроса, установите курсор на переключателе «в найденном» и нажмите левую клавишу «мыши». На переключателе будет отображен маркер красного цвета – признак того, что переключатель установлен.

Для получения всей базы сообщений в результате запроса время для поиска и текста для поиска не задается.

3.9.3 Сортировка сообщений

Сортировка сообщений выполняется среди сообщений, которые были получены в результате предыдущего запроса поиска.

Сортировка выполняется по одному из полей сообщения: «Имя задачи», «Сообщение», «Ошибки» или по хронологии – сортировать по времени.

Для выполнения сортировки выберите на панели сортировки из списка поля «Сортировать по полю» одно из возможных значений кроме значения  «нет» (рисунок 3.68).

Рисунок 3.68
